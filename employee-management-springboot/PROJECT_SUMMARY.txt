📦 PROJECT OVERVIEW
═══════════════════════════════════════════════════════════════════════════════
Your Java Swing Employee Management System has been fully converted to a
modern Spring Boot web application with a beautiful animated frontend!

🎯 KEY FEATURES CONVERTED FROM SWING:
───────────────────────────────────────────────────────────────────────────────
✅ Login Page          → Animated web login with particles & orbs
✅ Employee Table       → Paginated data table with sorting & filtering
✅ Add Employee         → Modal form with validation
✅ Update Employee      → Inline edit with confirmation
✅ Delete Employee      → Soft delete with confirmation dialog
✅ Search               → Real-time search with debouncing
✅ Navigation           → Collapsible sidebar navigation
✅ Status Bar           → Toast notifications & progress indicators

🚀 NEW FEATURES ADDED:
───────────────────────────────────────────────────────────────────────────────
✨ Role-Based Access Control (ADMIN/MANAGER/USER)
✨ Interactive Dashboard with Chart.js analytics
✨ Department & Designation management
✨ Salary distribution charts
✨ Employee growth trends
✨ Profile management page
✨ Reports & analytics page
✨ Responsive design for mobile/tablet
✨ Keyboard shortcuts
✨ Real-time notifications
✨ Demo accounts (admin/manager/user)

🚀 HOW TO RUN
═══════════════════════════════════════════════════════════════════════════════
1. Install Java 21 and Maven
2. Navigate to project folder
3. Run: mvn spring-boot:run
4. Open: http://localhost:8080

📡 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════
GET    /api/employees              → List all (paginated)
GET    /api/employees/{id}       → Get by ID
POST   /api/employees              → Create new
PUT    /api/employees/{id}       → Update
DELETE /api/employees/{id}       → Delete
GET    /api/employees/search     → Search
GET    /api/employees/stats      → Dashboard stats
